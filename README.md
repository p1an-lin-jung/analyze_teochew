# 潮州话-普通话拼音映射分析


基于潮州音字典(http://www.czyzd.com/)的数据，提取了近12,000条汉字的潮州话和普通话读音映射关系，分析潮州话与普通话之间的拼音（声母、韵母、声调）映射规律。

### 拼音映射文件
[pinyin_mapping.json](pinyin_mapping.json)

### 输出文件
普通话拼音映射到潮州话的统计：[普通话_to_潮州话.md](./普通话_to_潮州话.md)

潮州话拼音映射到普通话的统计：[潮州话_to_普通话.md](./潮州话_to_普通话.md)

### 数据规模
- **总映射数**: 12,059条
- **有效映射数**: 11,934条（过滤语气词后）
- **覆盖汉字**: 9,110个
- **多音字**: 1,304个
- **单音字**: 7,806个

## 🔄 数据处理流程

**处理规则**:
处理时，进行以下操作：

- 0、按照潮州府城音进行处理；

- 1、过滤掉`语气词、拟声词`等（发音多且不规律），去除`方言字`（没有对应普通话发音）

- 2、ü的标准化: `ju/qu/xu/yu/nue/lue`中的`u`转换为`v`；

- 3、零声母处理: 普通话拼音自动去除`y、w`零声母；

- 4、普通话的声调符号转换为数字：`xiàng -> xiang4`

- 5、分别按普通话映射到潮州话、潮州话映射到普通话

- 6、例字优先用`现代汉语3500常用字`

- 7、多音字处理成多条，以'相'为例，普通话有2音、潮州话有4音，对应关系如下:
```
{
    "char": "相",
    "mandarin_pinyin": "xiang1",
    "chaozhou_pinyin": "siang1"
  },
  {
    "char": "相",
    "mandarin_pinyin": "xiang1",
    "chaozhou_pinyin": "siê1"
  },
  {
    "char": "相",
    "mandarin_pinyin": "xiang4",
    "chaozhou_pinyin": "siang3"
  },
  {
    "char": "相",
    "mandarin_pinyin": "xiang4",
    "chaozhou_pinyin": "siên3"
  },

```


**统计内容**:
- 普通话22个声母 → 潮州话声母的映射
- 普通话5个声调 → 潮州话声调的映射
- 普通话47个韵母 → 潮州话韵母的映射（前30个）

**统计内容**:
- 潮州话18个声母 → 普通话声母的映射
- 潮州话8个声调 → 普通话声调的映射
- 潮州话81个韵母 → 普通话韵母的映射（前30个）
